# Установка в Mozilla Firefox

> Эта папка — **готовое расширение для Firefox**.

[← Вернуться к главной инструкции](../README.md)

---

## Установка

### 1. Скачай проект

GitHub → **Code** → **Download ZIP** → распакуй.

### 2. Открой отладку

В адресной строке:

```
about:debugging
```

Слева: **«Эта Firefox»**.

### 3. «Загрузить временное дополнение…»

### 4. Выбери файл `manifest.json`

Путь:

```
Proxy-plugin  →  firefox  →  manifest.json
```

Выбирается **файл** `manifest.json`, не папка.

---

## Готово

Иконка **G - Proxy** на панели. Нажми → вставь прокси → **+ Добавить прокси** → **▶** в списке.

---

## Важно

После **полного закрытия Firefox** расширение может пропасть — это нормально для «временного дополнения».  
Загрузи `manifest.json` снова (шаги 2–4).

Нужна Firefox **125+** (Меню → Справка → О Firefox).

---

## Обновление

1. Скачай новую версию  
2. `about:debugging` → загрузи новый `firefox/manifest.json`
