# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.9.0] — 2026-06-24

### Добавлено

- Список прокси с сохранением, переключением и редактированием
- Проверка IP, страны и пинга после добавления
- Режим «только для выбранных сайтов»
- Импорт и экспорт списка прокси (JSON)
- Отдельные сборки для **Chrome** и **Firefox**
- Русский интерфейс
- Поддержка HTTP, HTTPS и SOCKS5 (с авторизацией)

### Установка

1. Скачай архив для своего браузера из [Releases](https://github.com/Qerakl/Proxy-plugin/releases)
2. Распакуй
3. **Chrome:** `chrome://extensions` → режим разработчика → загрузить папку `chrome`
4. **Firefox:** `about:debugging` → загрузить `firefox/manifest.json`
