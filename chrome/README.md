# Установка в Google Chrome

> Эта папка — **готовое расширение**. Выбери её при установке в браузере.

[← Вернуться к главной инструкции](../README.md)

---

## Кому подходит

- Google Chrome  
- Яндекс.Браузер  
- Brave, Opera, Vivaldi  
- Microsoft Edge (режим расширений Chrome)

---

## Установка

### 1. Скачай проект

GitHub → **Code** → **Download ZIP** → распакуй.

### 2. Открой расширения

В адресной строке:

```
chrome://extensions
```

### 3. Включи «Режим разработчика»

Переключатель справа вверху.

### 4. «Загрузить распакованное расширение»

### 5. Выбери папку `chrome`

Путь выглядит так:

```
Proxy-plugin  →  chrome  →  (эта папка)
```

**Не** выбирай всю `Proxy-plugin` — только **`chrome`**.

---

## Готово

Иконка **G - Proxy** на панели. Нажми → вставь прокси → **+ Добавить прокси** → **▶** в списке.

---

## Обновление

1. Скачай новую версию с GitHub  
2. `chrome://extensions`  
3. Кнопка **Обновить** у G - Proxy или перезагрузи папку заново
